# ComunicacoesMoveis
PT - Este repositório trata da disciplia de Comunicações móveis ministrada pelo professor Vecente Sousa para o curso de Engenharia de Telecomunicações da UFRN. 

EN - This repository deals with the subject of mobile communications taught by Professor Vecente Sousa for the Telecommunications Engineering course at UFRN.

Este repositório está dividido em três projetos que foram elaborados e disponibilizados pelo professor, o primeiro projeto é formado por alguns desafios que, dentro da sua pasta, existem várias pequenas entregas que formam o projeto 01 completo.

E segue-se tanto para o projeto dois e três, a mesma lógica de organização, dividido em duas pastas, uma pasta com as figuras e os códigos utilizados e outra com o relatório elaborado para todo o projeto, conforme exigido no escopo do projeto.
