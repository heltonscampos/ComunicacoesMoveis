Este README tem como objetivo de descrever o que deve ser feito para executar o código da entrega 4 que é para mostrar o plot de um REM com macro e microcélulas.


======== Execução do código ==========

1 - Na MatLab (foi usada a versão R2015a), abra o arquivo handson_41.m que se encontra dentro desta página.
2 - Pode executar pressionando a tecla F5, ou o botão "RUN" dentro do editor.
3 - Aguarde um pouco que aparecerá as seis figuras com os REMs para cada um dos três cenários com e sem microcélulas.
