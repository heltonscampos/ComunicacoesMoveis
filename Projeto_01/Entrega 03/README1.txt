Este README tem como objetivo de descrever o que deve ser feito para executar o código da entrega 3 que é para demonstrar que o desvio padrão das amostras do sombreamento é constante correlacionado e igual ao desvio padrão de entrada independentemente da variação do coeficiente de correlação.


======== Execução do código ==========

1 - Na MatLab (foi usada a versão R2015a), abra o arquivo handson_41.m que se encontra dentro desta página.
2 - Pode executar pressionando a tecla F5, ou o botão "RUN" dentro do editor.
3 - Aguarde um pouco que aparecerá as três figuras com os REMs sem Shadowing, com Shadowing e com Shadowing correlacionado. Além disso, na IDE do Matlab, também foi plotado qual valor de dAlphacorr foi usado no cálculo do desvio padrão correlacionado e também em que ERB ele está realizando este cálculo.
