Instruções para rodar a calculadora LTE. Siga a instruções abaixo para executar corretamente a função da calculadora da taxa de transmissão da dados.

1 - Para executar a calculadora_LTE primeiramente deve-se possuir o software Matlab instalado em seu computador.
2 - Abra o arquivo Calculadora_LTE.m que encontra-se na pasta 'code'.
3 - Com o arquivo aberto no MatLab execute clicando em 'Run' ou pressionando F5.
4 - Selecione os parâmetros desejados no campo "parâmetros de entrada", clique em calcular e no campo "parâmetros de saída" será mostrado os valores para as taxas de transmissão que foram calculados.
